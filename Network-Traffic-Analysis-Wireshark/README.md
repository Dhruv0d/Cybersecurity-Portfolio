# Network Traffic Analysis and DNS Investigation using Wireshark

## Project Overview

Performed network traffic analysis using Wireshark to investigate a web browsing session captured in a packet capture (.pcap) file.

## Tools Used

- Wireshark
- TCP/IP
- DNS
- HTTP
- ICMP

## Skills Demonstrated

- Packet Analysis
- Protocol Analysis
- Threat Hunting
- Incident Investigation
- Network Monitoring

## Key Activities

- Analyzed DNS traffic
- Investigated TCP and HTTP communications
- Applied packet filtering using IP, MAC address, and port filters
- Examined packet metadata and protocol layers
- Conducted payload inspection for evidence gathering

## Outcome

Successfully traced a complete web communication session from DNS resolution through HTTP requests and extracted relevant network indicators.
