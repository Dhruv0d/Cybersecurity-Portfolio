 Phishing Incident Response and Alert Escalation

## Project Overview

Investigated a phishing alert involving a suspicious email attachment and performed incident triage using a structured phishing response playbook. Evaluated email indicators, analyzed attachment characteristics, validated threat intelligence, and documented findings to support escalation decisions.

## Objective

- Investigate a phishing security alert
- Identify indicators of compromise (IOCs)
- Validate malicious file indicators
- Apply a phishing response playbook
- Document findings and escalation decisions

## Key Activities

### Alert Investigation

- Reviewed phishing alert ticket details.
- Assessed alert severity and investigated supporting evidence.
- Followed a structured incident response workflow.

### Email Analysis

- Evaluated sender information and domain legitimacy.
- Identified phishing indicators including domain mismatch, spelling errors, and social engineering techniques.
- Assessed email content for signs of malicious intent.

### Attachment Investigation

- Investigated executable file attachment characteristics.
- Validated the attachment using SHA256 hash intelligence.
- Confirmed the file was malicious based on threat intelligence evidence.

### Escalation Decision

- Documented investigation findings.
- Determined escalation requirements based on playbook guidance.
- Updated the incident ticket with evidence-based recommendations.

## Outcome

Successfully investigated a phishing alert, identified multiple indicators of compromise, validated a malicious attachment through hash analysis, and escalated the incident according to defined SOC procedures. Demonstrated practical skills directly applicable to Security Operations Center (SOC) and Incident Response roles.
